# ☕ Coffee Shop Sales Dashboard

## 📌 Overview
This project is an interactive **Coffee Shop Sales Dashboard** created in **Microsoft Excel** to analyze sales performance, customer behavior, and product trends.  
The dashboard provides meaningful business insights using KPI cards, Pivot Tables, charts, and slicers.

---

## 📊 Dashboard Features
- 💰 Total Sales Analysis
- 👥 Customer Footfall Tracking
- 🧾 Average Bill Per Person
- 📦 Average Orders Per Person
- ⏰ Hourly Quantity Ordered Analysis
- 🥧 Product Category Distribution
- 🏪 Store-wise Sales & Footfall Comparison
- 🔥 Top 5 Products Based on Sales
- 📅 Weekday Order Trend Analysis
- 🎛️ Interactive Month & Day Filters
